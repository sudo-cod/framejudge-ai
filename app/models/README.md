# Bundled fingerprint model

`mobilenetv2-7.onnx` is the MobileNetV2 model from the ONNX Model Zoo mirror:

https://huggingface.co/onnxmodelzoo/mobilenetv2-7

The upstream ONNX Model Zoo is distributed under the Apache License 2.0. The
model runs locally through ONNX Runtime. No video frames are sent to a cloud AI
service.
